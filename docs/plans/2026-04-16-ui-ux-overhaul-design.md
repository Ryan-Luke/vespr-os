# VESPR UI/UX Overhaul Design

**Date:** 2026-04-16
**Status:** Approved
**Direction:** Stripe/Notion — warm, approachable, premium
**Palette:** Warm stone neutrals + teal accent (surgical use)
**Density:** Balanced (Stripe-style)

## Color System
- Base: #0C0A09, Raised: #1C1917, Card: #292524, Elevated: #44403C
- Text: #F5F5F4 primary, #A8A29E secondary, #78716C muted
- Accent: #14B8A6 (teal-500), hover #0D9488, muted 10% opacity
- Borders: rgba(255,255,255,0.06) default, 0.10 hover
- Status: green success, amber warning, red error, teal active

## Typography
- Headings: Inter 600, -0.025em tracking
- Body: Inter 400, 14px, line-height 1.5
- Labels: Inter 500, 12px, uppercase, 0.05em tracking
- KPIs: Inter 700, 28px, tabular-nums

## Components
- Cards: #1C1917 bg, 1px border rgba(255,255,255,0.06), 12px radius, no shadow
- Sidebar: 256px, teal active indicator (3px left border + 8% bg)
- Buttons: teal primary, transparent secondary, 8px radius
- Modals: stone-900, 16px radius, blur(8px) overlay, spring animation
- Chat: flat messages (no bubbles), avatar+name+content layout

## Animations
- 150ms ease hovers, 200ms spring modals, 30ms list stagger
- No decorative animations
